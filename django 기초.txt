가상환경  만들기
1. C:\에 파일 생성
1-1. C:\>md 파일명1  ->  파일 생성
2.C:\파일명1\>python -m venv 파일명2
3.C:\파일명1\파일명2\Scripts\>까지 입력했다면
이어서 .\activate 입력
4. (mywebsite) PS C:\test\mywebsite\Scripts>가 뜨면 가상환경 완성!

가상환경 지우기
-deactivate 입력!


Django 프레임워크 입문

프로젝트 생성 명령어
: django-admin startproject 프로젝트명

앱 생성 명령어
: django-admin startapp  프로젝트 內, 앱명

서버구동
: (프로젝트 생성 후, 프로젝트 내부에서)
 py manage.py runserver (포트 번호(無->기본 8000번 포트))

기본 테이블 생성 및 superuser 생성
: (프로젝트 생성 후, 프로젝트 내부에서)
기본 테이블 생성: py manage.py migrate
ㄴ(db.sqlite3라는 파일이 생깁니다. 이 파일이 바로 데이터베이스)
슈퍼유저 생성: py manage.py createsuperuser
ㄴ(이후에 사이트에 접속하고...)
관리자 모드 접속: localhost:8000/admin를 주소창에 입력
